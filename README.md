# InSight-Website


A household financial management app that helps users track bills, monitor monthly spending, and receive saving tips when overspending is detected. Supports two roles: household owner and admin.

## Features
- Add, view, and search bills by month, year, or status
- Monthly overview with overspending detection
- Personalized saving tips for overspent months
- Admin panel for platform management
- Secure login and registration

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js
- Database: MySQL
